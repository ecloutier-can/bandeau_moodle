# Générateur de bandeau Moodle

Un outil simple et efficace pour générer des bandeaux personnalisés pour vos cours Moodle.

## 🚀 Accéder à l'application

L'application est consultable en ligne ici :
👉 **[https://ecloutier-can.github.io/bandeau_moodle/](https://ecloutier-can.github.io/bandeau_moodle/)**

## ✨ Fonctionnalités

- Personnalisation du titre du bandeau.
- Sélection de palettes de couleurs prédéfinies ou personnalisées.
- Ajout dynamique de paragraphes de contenu.
- Génération instantanée du code HTML prêt à être collé dans Moodle.
- Aperçu en temps réel du résultat final.

## 🛠️ Utilisation dans Moodle

1. Configurez votre bandeau dans l'application.
2. Cliquez sur **Copier le code HTML**.
3. Dans Moodle, ajoutez une activité ou une ressource (type "Texte et média").
4. Activez l'affichage du code source (`</>`) dans l'éditeur.
5. Collez le code et enregistrez.
